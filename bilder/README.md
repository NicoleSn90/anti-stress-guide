# Bilder

In diesen Ordner kommen die Bilder für die Website (z. B. die Hunderassen-Fotos).

## So fügst du Bilder ein
1. Lege deine Bilddateien hier ab, am besten mit klaren Namen, z. B.:
   - `labrador.jpg`
   - `schaeferhund.jpg`
   - `pudel.jpg`
2. Im HTML werden sie dann so eingebunden:
   ```html
   <img src="bilder/labrador.jpg" alt="Labrador Retriever" />
   ```

## Tipps
- **Format:** `.jpg` für Fotos, `.png` für Grafiken mit Transparenz, `.webp` für kleine Dateigröße.
- **Größe:** Bilder vor dem Hochladen auf eine sinnvolle Web-Größe verkleinern (z. B. max. 1200 px breit), damit die Seite schnell lädt.
- **Namen:** nur Kleinbuchstaben, keine Leerzeichen/Umlaute (z. B. `franzoesische-bulldogge.jpg`).
